## 🧑‍💻 About the Project

This project is my personal portfolio site, designed to present my projects, technical skills, and development experience.

The site features dynamic project cards, smooth animations, and responsive layouts to create a modern and engaging browsing experience.

It highlights projects that demonstrate both frontend development and backend architecture, including API integrations and database-driven applications.


## ⚙️ Tech Stack

Frontend:
- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion

Backend:
- REST APIs
- Node.js
- PostgreSQL

Tools:
- Git
- GitHub
- Vercel


## ✨ Features

- Interactive project cards with expandable project details
- Smooth UI animations
- Fully responsive design
- Modular component architecture
- API-driven functionality
- Optimised performance using modern React features

src

## 🛠️ Getting Started

Clone the repository and install dependencies:
    git clone https://github.com/SobiahSelvarajah/ss-my-portfolio
    cd ss-my-portfolio
    npm install
    npm run dev

Open the development server:
http://localhost:3000
